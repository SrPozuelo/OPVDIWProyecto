# BOOTSTRAP: INVESTIGACIÓN E IMPLEMENTACIÓN
>El objetivo principal de este proyecto es la investigación del framework Bootstrap para maximizar la eficiencia en el desarrollo frontend, garantizando la compatibilidad entre navegadores, optimizando la experiencia de usuario a través de componentes interactivos con una arquitectura de información limpia y accesible y desarrollar como ejemplo una pequeña aplicación web moderna y totalmente adaptable (Responsive Design).

## Tecnologías utilizadas:
1. Boostrap v5.3.2
2. Visual Studio Code 1.122.1
3. Git
4. GitHub

## Objetivos del proyecto:
>1. Implementación de un Diseño Responsive: Utilizar el sistema de rejilla (Grid System) de Bootstrap para garantizar que la interfaz se adapte correctamente a dispositivos móviles, tablets y escritorio.

>2. Dominio de Componentes: Integrar y personalizar componentes complejos como barras de navegación (navbars), tarjetas (cards), carruseles, modales y formularios validados.

>3. Optimización de Estilos: Personalizar la apariencia predeterminada de Bootstrap mediante la sobreescritura de CSS o el uso de variables Sass para dar una identidad visual única al proyecto.

>4. Accesibilidad Web: Asegurar que el proyecto cumpla con estándares básicos de accesibilidad (uso de etiquetas ARIA, contrastes adecuados y navegación por teclado).

>5. Eficiencia en el Desarrollo: Demostrar cómo el uso de un framework reduce los tiempos de producción frente al CSS puro (Vanilla CSS).

>6. Documentación Técnica: Redactar una memoria que detalle la estructura de archivos, las dependencias utilizadas (como iconos de FontAwesome o Google Fonts) y la lógica del diseño.
